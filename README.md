# 🐳 Docker + Node.js Demo App

This is a simple Node.js + Express application containerized using Docker. It demonstrates how to create, run, and manage a lightweight web API inside a Docker container.

---

## 📦 Tech Stack

- **Node.js**
- **Express.js**
- **Docker**

---
